Battery Data Format Application Ontology Terms#

Battery Data Format#

ambient_pressure_pascal#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#ambient_pressure_pascal

Comment the ambient pressure in unit Psacal

ambient_temperature_celsius#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#ambient_temperature_celsius

Comment the ambient temperature in degree Celsius

current_ampere#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#current_ampere

Comment Instantaneous current, recorded in amperes

cycle_dimensionless#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#cycle_dimensionless

Comment The cycle number here is used to segregate data into smaller quasi-periodic subsets, and is typically used to track evolution of performance metrics over the course of aging.

dcir_ohm#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#dcir_ohm

Comment direct current internal resistance in unit ohm

step_dimensionless#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#step_dimensionless

Comment the step number with unit one

surface_pressure_pascal#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#surface_pressure_pascal

Comment the surface pressure in unit Psacal

surface_temperature_celsius#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#surface_temperature_celsius

Comment the surface temperature in degree Celsius

test_time_millisecond#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#test_time_millisecond

Comment the number of milliseconds since the start of the test.

voltage_volt#

IRI: https://w3id.org/battery-data-alliance/ontology/battery-data-format#voltage_volt

Comment Instantaneous voltage, recorded in volts
